# Around The U.S. – Versión con Autenticación

## Descripción

Este proyecto es una aplicación web desarrollada con React donde los usuarios pueden registrarse, iniciar sesión y interactuar con una interfaz de tarjetas de imágenes. Después de autenticarse, los usuarios pueden ver tarjetas, darles "like", eliminar sus propias tarjetas y actualizar la información de su perfil.

La autenticación se maneja mediante una API externa que devuelve un JSON Web Token (JWT). Este token se guarda en localStorage para mantener la sesión del usuario activa.

## Tecnologías utilizadas

- React
- JavaScript (ES6+)
- React Router
- CSS
- REST API
- JSON Web Tokens (JWT)

## Funcionalidades

- Registro de usuarios
- Inicio de sesión y autenticación
- Rutas protegidas para usuarios autenticados
- Edición del perfil
- Agregar nuevas tarjetas
- Eliminar tarjetas propias
- Dar y quitar "likes" a tarjetas
- Persistencia de sesión mediante localStorage
